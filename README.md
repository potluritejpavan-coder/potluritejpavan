## Lost & Found

### Features
- Easily report lost items.
- Track lost items in specific locations.
- User-friendly interface for reporting and viewing lost items.

### Usage Instructions
1. Navigate to the "Lost & Found" section in the app.
2. Click on "Report Lost Item" to submit details.
3. Check the status of your lost items within the same section.

### Lost Locations
- Main Campus: Front Desk, Lost & Found Office
- Sports Complex: Security Office
- Library: Information Desk
